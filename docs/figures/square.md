# Квадрат

## Функции

- `area(a: float) -> float`

  Рассчитывает площадь квадрата из его стороны.

    - a - Сторона квадрата

  Возвращает площадь квадрата

- `perimeter(a: float) -> float`

  Рассчитывает площадь квадрата из его стороны.

    - a - Сторона квадрата

  Возвращает периметр квадрата

## Примеры использования

```python
print(area(5))  # 25
print(perimeter(5))  # 20
```
